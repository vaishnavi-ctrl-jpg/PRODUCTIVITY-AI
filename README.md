# Productivity AI — Personal AI Assistant

A multi-agent AI productivity system built with Google ADK and Gemini 2.0 Flash.

🔗 **Live:** https://productivity-ai.onrender.com

## Features
- Task management with priorities and due dates
- Smart notes with AI summarization
- Wikipedia research
- Daily briefing
- Session memory per user
- Beautiful dark chat UI

## Stack
- Google ADK + Gemini 2.0 Flash
- FastAPI + Uvicorn
- Deployed on Render

## Run locally
```bash
pip install -r requirements.txt
cp .env.example .env  # add your API key
python main.py
# Open http://localhost:8080
```
